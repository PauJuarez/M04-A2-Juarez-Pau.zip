Git Tutorial Summary

By: Aleix Calle Ramon

  1. Git Installation:
    • Download and install Git from git-scm.com.
    • Verify the installation by running git --version in the command line.
  2. Initial Configuration:
    • Set up your username and email address with: 
      git config --global user.name "Your Name"
git config --global user.email "your@email.com"
  3. Creating a New Repository:
    • Initialize a new repository in an existing directory with: 
      git init
  4. Cloning an Existing Repository:
    • Clone a remote repository to your local machine with: 
      git clone <repository_url>
  5. Adding and Committing Changes:
    • Add files to the staging area with: 
      git add <file_name>
    • Commit changes with a descriptive message: 
      git commit -m "Descriptive message"
  6. Checking the Status of Changes:
    • Check the status of files with: 
      git status
  7. Syncing with a Remote Repository:
    • Push your changes to the remote repository with: 
      git push origin <local_branch>
    • Fetch changes from the remote repository with: 
      git pull origin <local_branch>
  8. Branching:
    • Create a new branch with: 
      git branch <branch_name>
    • Switch to an existing branch with: 
      git checkout <branch_name>
  9. Merging Branches:
    • Merge a branch into the current branch with: 
      git merge <branch_name>
  10. Conflict Resolution:
    • If there are conflicts during the merge, resolve them manually and then commit the changes.
